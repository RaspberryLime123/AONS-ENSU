[Install]
=========
- Windows:  Download and install MaySkySetup[insani][AT2008].exe.
- Mac OS X: Download MaySkyMac[insani][AT2008].zip and unarchive it, then move
            the application bundle wherever you wish.

[Uninstall]
===========
- Windows:  Use the provided uninstaller.
- Mac OS X: Simply delete May Sky.app.

[Basic Operations]
==================
Left-clicking, enter, and spacebar will allow you to progress to the next line of text.
Right-clicking will bring up the save, load, and entering log mode options.
The left arrow will also bring you into log mode.

[Full Screen]
=============
Pressing the "f" key will change the game to full screen mode.
